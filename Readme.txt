This project contains gut microbiota data of thyroid nodule and controls.
Basic information
# ThyroidNodule.N10.specy.txt
Microbial composition of all cases.
# ThyroidNodule.N10.pathway.txt
Functional composition of all cases.
# ThyroidNodule.Metadata
Contains obs, generichness, age, BMI, gender and group.


Thyroid Nodule/
# TN.specy.sig.txt
MASSLIN2 result of microbiota and age, BMI ,gender and thyroid nodule group.
# TN.pathway.sig.txt
MASSLIN2 result of pathway and age, BMI ,gender and thyroid nodule group.

Thyroid Function/
# FT3.specy.sig.txt
MASSLIN2 result of microbiota and age, BMI ,gender group and thyroid functions(FT3,FT4,TSH).
# FT3.pathway.sig.txt
MASSLIN2 result of pathway and age, BMI ,gender and thyroid functions(FT3,FT4,TSH).
